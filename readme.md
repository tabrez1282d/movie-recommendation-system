# 🎬 Movie Recommendation System

It is a hybrid movie recommendation system built using the MovieLens ml-25m dataset.

## 🔍 How It Works
Combines two techniques:
- **Collaborative Filtering** — recommends based on user ratings
- **Content Based Filtering** — recommends based on movie genres
- **Final Score** = 70% rating similarity + 30% genre similarity

## 📦 Dataset
- [MovieLens ml-25m](https://grouplens.org/datasets/movielens/25m/)
- 62,423 movies and 25 million ratings
